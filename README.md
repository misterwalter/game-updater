# game-updater
Does what it says on the tin, simply and interactively

Don't forget to replace the default steam account name with your steam username.

`alias gu="python ~/code/game-updater/game-updater.py"` in the old .bashrc makes life _even_ easier. Once you've got that, just navigate to the godot project folder and run it once. it'll fill you in on what you need, and once you do that, you can run it again and it'll probably work!
